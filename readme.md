# AI Benchmark System

A benchmark system for evaluating large language models on multi-step reasoning and instruction-following tasks.

## Project Structure

- `data/raw/` - Raw task datasets
- `data/processed/` - Processed datasets
- `src/` - Source code modules
- `notebooks/` - Jupyter notebooks for analysis
- `results/` - Benchmark results and metrics
- `configs/` - Configuration files
